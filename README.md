## 🎮 ゲーム紹介

TileOut（タイルアウト）は、盤面上のタイルをスライドさせて戦う、シンプルながら奥深い戦略ボードゲームです。
互いに異なる移動制限を持つプレイヤー同士が、知略を巡らせて勝利を目指します。

## 📜 ルール概要

このゲームは2人のプレイヤーで対戦します。

- **Player 1 (Vertical)**: 列を**上**にのみスライドできます。
- **Player 2 (Horizontal)**: 行を**右**にのみスライドできます。

### 🏆 勝利条件
自分の色のタイルを **10個** 連結させれば勝利です！
相手の動きを予測し、妨害しながら自分のラインを完成させましょう。

## ✨ 特徴
- **スライド戦略**: 単純な移動ルールが生み出す無限の戦略。
- **直感的な操作**: タッチ/クリックで直感的にスライド操作が可能。
- **心理戦**: 相手がどこを動かしたいか？先を読む力が試されます。
